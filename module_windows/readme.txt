HOW TO INSTALL PDFX ON WINDOWS

Step 1 : Install Python 2.7.

Step 2 : Download pdftk from this link and install it.
	----------------------------------------------------
	https://www.pdflabs.com/tools/pdftk-the-pdf-toolkit/
	----------------------------------------------------

Step 3 : Run installer.py (make sure you have an active internet connection.)

Step 4 : Run main.py and enter the name of patent file(without extension)
